# Advanced Mapping

The MUME client features a powerful, automated mapping system.

## 🧭 Smart Navigation
- **Hold to Walk**: Long-press any room on the map to automatically walk there. The character will take the shortest path.
- **Sequential Movement**: Commands are sent one-by-one only after the client confirms you have changed rooms.
- **Release to Stop**: Simply lift your finger or mouse button at any time to stop movement safely.
- **Standard Tap**: Tap a room once to view its name, area, and terrain info.

## 🔍 Visual Indicators
- **Quest Flags**: Rooms marked with a `?` indicate potential quest locations or points of interest.
- **Vertical Exits**: Small arrows (▲/▼) indicate if a room has an up or down exit.
- **Pan & Zoom**: Click/drag to pan. Pinch or use the scroll wheel to zoom. 

## 🏗️ Design Mode
Enable "Design Mode" in the mapper toolbar to manually edit the map:
- **Add Room**: Long-tap on empty space to add a custom room.
- **Edit Details**: Select a room to change its name, description, or terrain type.
- **Sync Location**: Use the context menu to force-sync your location to a specific coordinate.

## 📥 External Data
The client supports `.mm2` files from MMapper. Use the mapper dropdown to import your existing world data for immediate navigation across Middle-earth.
